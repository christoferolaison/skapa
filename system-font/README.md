# `@skapa/system-font`

> System font

## Install

```bash
  yarn add @skapa/system-font
```

```bash
  npm install @skapa/system-font
```

## Usage

```javascript
const systemFont = require('@skapa/system-font')
// or
import systemFont from '@skapa/system-font'

const styles = {
  fontFamily: systemFont,
}
```
